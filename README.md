flask-tutorial
==============

flask-tutorial

Install:
``` bash
$ sudo apt-get install python-pip python-dev build-essential 
$ sudo pip install --upgrade pip 
```

``` bash
$ sudo apt-get install python-virtualenv
```

Create an environment/project:
```bash
$ mkdir myproject
$ cd myproject
$ virtualenv venv
```

```bash
$ . venv/bin/activate
```

```bash
$ pip install flask
```
